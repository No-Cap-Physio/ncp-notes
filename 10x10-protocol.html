The 10×10 Protocol

Everything Derek Hansen has published on the 10×10 repeat sprint protocol — reconstructed, extended, and reframed for physiotherapists and S&C coaches working with ACL, post-surgical knees, ankles, and hip injuries.

ACL Rehab Ankle Return to Run Knee Surgery Sprint Training S&C
00

The 30-Second Version

Ten 10-meter sprint accelerations done back to back, repeated for multiple sets. You sprint 10m, decelerate, turn, sprint back. That is one rep. Ten reps is one set. Each set covers 100m total.

That is the whole thing. The simplicity is the point.

Why you care
  • Cleanest framework for safely re-loading running after lower limb injury
  • You control distance, density, and intensity with cones and a stopwatch
  • No fancy gear required — works in a 25m hallway or on a full field
  • Same protocol runs rehab, conditioning, and acceleration development
01

Origins

  • Gerard Mach (Poland) — original architect. Understood keeping fitness during injury beats complete rest.
  • Charlie Francis and Dr. Douglas Clement (Canada, 1970s) — popularised it for hamstring strains in track and field.
  • Derek Hansen — spent 35+ years developing it from a single rehab tool into a full performance system.
  • No MRI, no ultrasound. Built on basic biomechanics and common sense. Survived because it works.
The original math
  • Sprinter doing 6 × 100m = 600m per session
  • Injured athlete can't run 100m, so: 60 × 10m = 600m same volume
  • Organised as 6 sets × 10 reps or 10 sets × 6 reps — both work
  • Far lower tissue risk, fitness preserved, return to full training almost seamless
02

Why It Actually Works — 5 Mechanisms

  1. Low eccentric tissue load. In a 10m sprint (~6–7 steps) the body stays in forward-lean acceleration posture. Foot strikes while knee is still bent. Limits hamstring lengthening under load. Strength stimulus without dangerous stretch. Adjust posture and same logic protects other tissues.
  2. Volume without peak velocity. Density keeps intensity sub-maximal. Stack 40–60 reps per session — impossible at full sprint. A 10m acceleration reaches ~70–75% of max velocity, which is actually a meaningful speed in sport contexts (game average is <50% max velocity for 80–85% of play).
  3. Technique reps. Sub-maximal conditions = best environment to groove mechanics. Dozens of chances to clean up posture, shin angle, arm action until it's automatic.
  4. General supercompensation. Density creates a broad fitness lift covering speed, strength, and endurance simultaneously. Upper body, core, glutes, hamstrings, calves, feet — total body stimulus.
  5. Athletes return sharper than healthy teammates. Injured athlete doing 4 sets of 10×10 daily = 40 near-max accelerations per day. Healthy teammates jogging, walking, standing in practice. The rehabbing athlete comes back fitter.
Evidence of strength gains
  • Athletes reported higher Nordboard scores after just 2 weeks of 10×10 hamstring rehab
  • This confirms the protocol is genuinely strengthening the hamstring, not just maintaining it

The Three Dials — One Structure, Three Outcomes

Change the work-to-rest ratio to flip the goal. Same cones, same distance, different adaptation.

Conditioning
  • Set time: ~60s
  • Rep interval: ~6s
  • HR peaks: 180–190bpm
  • Effort: sub-max, continuous
  • Goal: work capacity + aerobic base
Rehab / RTP ← home base
  • Set time: 90–120s
  • Rep interval: 10–12s
  • Effort: sub-max with high coach/athlete awareness
  • Goal: safe re-loading + tissue tolerance
  • Density high enough to build capacity, velocity low enough to protect
Speed / Acceleration
  • Set time: 150–200s
  • Rep interval: 18–30s
  • Effort: maximal, no drop-off allowed
  • Goal: repeat sprint ability + acceleration quality
  • All 10 reps at identical quality
Morphing the protocol
  • You can restructure entirely: 5×20, 4×25, 7×15
  • Target ~100m per set regardless of rep/distance combo
  • As athletes progress: 15×7, 20×5, 25×4, 30×4 (~120m)
  • MLB pitchers used it timed to the pitch clock — that's how flexible it is
05

Start Type Modifications

Hansen uses four distinct start positions to load the 10×10 differently. Each changes the eccentric demand, force angle, and CNS intensity. Progress through these as tissue tolerance improves.

Start Type 1 · Lowest load

High Falling Start

  • Position: tall upright stance, lean forward from ankles, fall into sprint
  • Rep interval: 8–9s
  • Set completion: 80–90s
  • Best for: early rehab, ankle/calf/hip flexor injuries, onboarding, athletes with poor deceleration control
  • Load: lowest of the four — least CNS demand, most protective
Start Type 2 · Moderate load

Low Falling Start

  • Position: deeper squat-like position before the fall — hands NOT touching the floor
  • Rep interval: 12–40s (wide range depending on load target)
  • Set completion: 120–140s
  • Best for: intermediate rehab stage, building posterior chain activation, higher knee load
  • Load: moderate — deeper hip/knee flexion increases demand
Start Type 3 · High load

Three-Point Start

  • Position: one hand down, track-style set position
  • Rep interval: ~18s
  • Set completion: ~180s
  • Rest between sets: 2–5 min
  • Best for: late-stage RTP, performance phase, athletes needing maximal acceleration mechanics
  • Load: high — maximum hip flexion and posterior chain demand at start
Start Type 4 · Maximum load

Push-Up Start

  • Position: begin in push-up position on the ground, explode into sprint
  • Rep interval: ~20s
  • Set completion: ~200s
  • Rest between sets: 2–5 min
  • Best for: GPP performance work, highly conditioned athletes, sport-specific reaction training
  • Load: maximum — most demanding start position, requires full CNS recovery between reps
How to use these in rehab
  • Ankle / calf / hip flexor: start with High Falling Start (upright posture), build into Low Falling Start as tissue tolerates more load
  • Knee / ACL: begin with High Falling Start once criteria met, progress to Low → Three-Point as quad symmetry improves
  • Hamstring: can progress faster through the continuum — acceleration posture is already protective for the hamstring
  • Do not rush start type progression — each start significantly changes the demand on the system

High-Low Programming — Where 10×10 Sits

The High-Low concept from Charlie Francis gives 10×10 its programming context. Knowing where it sits on the intensity spectrum is what lets you use it intelligently across a full week.

The Three Zones

ZoneVelocityEffectWhere 10×10 sits
High90–100%Direct CNS adaptation, maximal recruitment, speed gains. Requires 48–72h recovery.Speed dial (18–30s rest/rep) — top end
Medium75–90%Significant peripheral and central fatigue. High injury risk. Impairs recovery. Francis avoided this zone for sprinters.Avoid — do not get stuck here
Low<70–75%Circulatory recovery, capillary density, reduces injury risk. Does not directly build speed but supports it.Conditioning dial (6s/rep) sits near the top of low
Where 10×10 lives on the spectrum
  • A 10m rep reaches ~70–75% max velocity — puts it at the top end of Low / entry of High depending on rest
  • Short rest (conditioning dial): sits comfortably in Low — high volume, recoverable, safe
  • Long rest (speed dial): pushes into High — CNS involvement, quality reps, requires 48–72h recovery
  • This is why the protocol can run on consecutive days (conditioning/rehab) but speed-dial use needs more recovery

Weekly Structure for Rehab

  • Consecutive days: Hansen runs 3–4 loading days in a row, then 1 recovery day, then repeat
  • Intensity progression across days: Day 1 at 60% → Day 2 at 70–75% → Day 3 at 75% → Day 4 at 75–80%
  • Volume stays constant day-to-day (e.g., 3 sets) while intensity naturally rises as athlete feels better
  • If you also increase volume AND intensity simultaneously, overload risk grows fast
  • Recovery day: upper body weights or low-intensity work only

High-Low Ratios by Sport

Francis's data on elite sprinters: 35% high-intensity, 65% low-intensity. For team sport athletes the balance shifts further toward low. For rehab populations, the 10×10 conditioning dial does most of the heavy lifting in the low zone.

GPP Phase Note

  • In GPP, intensity stays relatively flat day to day — less dramatic high-low swings
  • Build general base first. Polarise later (specific prep, competitive phase)
  • Short-changing GPP kills your peak when it matters most
  • The 10×10 in conditioning mode is perfect GPP work — high volume, low CNS stress, builds fitness foundation
The medium zone warning
  • Avoid sustained work at 75–90% velocity — this is the glycolytic grind zone
  • High fatigue, high injury risk, poor recovery, and no direct speed gains
  • If your 10×10 session leaves athletes in a 400m-race-style pit of despair, you're in the medium zone — reduce density
07

Monitoring

You do not need tech. But if you have it, here is the hierarchy.

Tier 1 — Stopwatch (always)
  • Time the full set of 10 reps end to end
  • Track across sessions — stable or improving set times = capacity building
  • Use a countdown (3-2-1-go or whistle) to control work-to-rest precision
Tier 2 — Timing Gates
  • Capture individual 10m times rep by rep — aiming for ~1.6s per rep
  • Key signal: reproducibility across the full set
  • Well-conditioned athlete: holds times across 7–8 reps before minimal drop-off
  • Underprepared athlete: 1–3 good reps then 20–25% drop-off
  • Target: less than 10% drop-off across the full set
  • Limitation: requires one athlete at a time — impractical for large groups
Tier 3 — GPS
  • Captures velocity, acceleration, and deceleration across large groups simultaneously
  • Set velocity thresholds to keep intensity honest across the session
  • Can compare sprint test data against 10×10 performance over time
Tier 4 — Moxie Monitor (advanced)
  • Local muscle oxygenation monitoring — tracks how fast O₂ depletes in the working muscle
  • Rapid depletion = muscle fatiguing quickly, possibly not ready for full loading
  • Useful injury monitoring tool in early RTP stages
Using times as a rehab speed governor
  • Get the athlete's best 10m time from a sprint test first
  • In early rehab, cap them at 50% slower than that time
  • Example: best time = 1.6s → rehab start = no faster than 2.3–2.4s
  • Each session, shave off 5–10% until they're back at full velocity
  • This is the single best way to prevent athletes running too fast too early

Always video sessions. Action camera on the side gives you a coaching review layer that real-time supervision misses, especially with larger groups.

  • Move up in 2.5m steps for ~5% jumps — gentler ramp for fragile populations

Option A: Volume Loading (same distance)

  • Stay at 10m and simply add sets: 3 → 4 → 5 → 6
  • Best for: linemen, goalkeepers, athletes who never need to run >10m in sport
  • Add sets, get faster with same distance — still builds tissue tolerance

Option B: Short-to-Long Distance Progression

  • 10×10 → 15×7 → 20×5 → 25×4 → 30×4
  • Best for: wide receivers, basketball players, strikers, track athletes
  • Combines volume and velocity increase across the block
09

Variations Worth Knowing

  • Curved runs: accelerate on a shallow arc, progress to deeper arcs. Minimum curve radius: ~9–10 yards (FIFA centre circle size). Loads lateral stability through foot, ankle, knee, hip, core. Test both directions — common to find asymmetry. Limit sets — cumulative stress is real.
  • Shortened deceleration zones: progress decel zone 10m → 7m → 5m. Advanced athletes can hit 3m max (limit to 2 sets). Builds eccentric braking strength. Directly relevant to cutting and landing tolerance.
  • Shuttle variation: 3 × 10yd back-to-back (there-back-there) then a longer break of 15–20s. Each "super-rep" = ~30yd. Total set still ~100–120yd.
  • Sport-specific loading: dribble a basketball through reps. Carry a racket, lacrosse stick, or football. Start on grass → progress to track (faster surface). Start in runners → progress to cleats or court shoes. Same protocol, increased transfer.
  • 5m start option: if 10m feels too aggressive, start at 5m. Keep reps at 10–12 (not 20 — too many reps per set).
The discipline warning
  • Teams consistently add sleds, vests, extra cones, "creative" twists
  • Hansen: "I give them something simple and they send me back a video where they've changed it"
  • Every addition dilutes the core stimulus. Get the clean reps in first.
  • Variations are for later stages, not for making early sessions "more interesting"

Injury-Specific Application

Hansen built this for hamstrings. Your population is knees, ankles, and hips. The structure transfers perfectly, but two things must change per injury: posture and entry gate.

Posture by Injury

InjuryPostureStart type to beginReason
HamstringForward-lean, stay lowLow Falling → Three-PointKeeps knee flexed, limits hamstring stretch
Knee / ACLAcceleration posture, short distance firstHigh Falling → Low FallingManages knee load, limits early velocity
Calf / Achilles / foot / ankleMore upright, jog-in or high-knee lead-inHigh Falling (tall/upright)Shifts demand off distal tissue, eases into load
Hip flexor / adductorMore upright, gentle build-in, front-side runningHigh Falling (tall/upright)Reduces hip flexion load, keeps leg more in front of body
Rule of thumb
  • Knee and hamstring → acceleration posture (forward lean, hard start)
  • Calf, Achilles, ankle, hip flexor → upright posture with a rolling start or high-knee jog-in
  • Once healthy, all injuries can progress to harder acceleration postures and start types
  • Trial, error, and athlete feedback dials in the exact posture. Use video to check it.

Entry Gates — Do Not Start Running Until These Clear

GateACL / Post-Surgical Knee
  • Quad strength is the gatekeeper, not the calendar. Non-negotiable.
  • Gate to begin running: quad limb symmetry index (LSI) ≥70% on surgical side
  • Target before loading running hard: ≥80% quad symmetry (below this, knee loading patterns deviate from healthy controls)
  • Strength benchmark for successful run initiation: ~1.45 Nm/kg bodyweight
  • Earliest realistic running window: ~12 weeks / 3–4 months post-op — earliest possible, never the trigger
  • Also need: full/near-full ROM, no swelling, normalised gait, clean single-leg control
  • For cutting/COD later: push to ≥85% → ≥90–95% + hop battery + psychological readiness
  • 9 months minimum to return to sport. Every month of delay up to 9 months lowers re-injury risk. Single biggest lever on outcomes.
  • Re-injury rate if criteria passed: 5.6%. If criteria failed: 38.2%.
GateAnkle — lateral sprain most common
  • No swelling, minimal/no pain, full ROM restored
  • Tolerance checks before running: single-leg balance, single-leg calf raise near symmetry, ~20s toe stand, ~10 single-leg hops pain-free
  • Progression: straight jog → linear sprint (High Falling) → arc/curved runs (shallow → deep) → planned COD → reactive agility
  • Arc running is the bridge between straight-line and cutting. Do not skip it.
  • Expose the ankle to its first compromised, loaded, reactive position in your controlled setting, not in a live game
  • Grade 1: return to activity in 1–3 weeks. Grade 2: 3–6 weeks. Grade 3: 4–6+ weeks.
GateHip Flexor / Adductor
  • Upright posture, gentle jog-in lead, leg stays more front-side
  • Progress distance conservatively — same short-to-long logic
  • Watch for groin/front-of-hip irritation after session — your dose monitor

The NCP Return-to-Run System

Your repeatable framework. Same skeleton for every lower limb injury. Swap posture, start type, and entry gate per injury.

GateClear Before Any Running
  • Injury-specific strength symmetry threshold met (Section 10)
  • No swelling at rest or after low-impact activity
  • Acceptable pain level (≤2/10 during activity)
  • ROM restored to functional range
  • Athlete psychologically ready
Phase 1Re-introduce — Volume, Low Velocity
  • Distance: 10m reps
  • Posture: injury-appropriate (upright for ankle/calf/hip; low for hamstring/knee)
  • Start type: High Falling Start
  • Intensity: sub-max, 60–70%, time-governed (cap at 50% slower than known sprint PB)
  • Sets/reps: 3–4 × 10, ~6–8s per rep, ~90–120s per set
  • Rest: 3 min between sets
  • Surface: flat, straight, consistent (grass or track)
  • Exit when: pain-free, no swelling response next morning, mechanics clean on video, stable set times
Phase 2Build — Velocity + Decel Demand
  • Distance: progress 10m → 15m → 20m in 2.5–5m steps
  • Start type: progress to Low Falling Start
  • Intensity: 75–85%
  • Sets/reps: 4 × 10
  • Begin shortening decel zone: 10m → 7m → 5m
  • Add shallow arc runs — this is your COD primer
  • Exit when: tolerates higher velocity, decel clean on video, symmetry target rising, no post-session symptom flare
Phase 3Express — Multidirectional + Sport-Specific
  • Distance: 20–30m, near top velocity
  • Start type: progress to Three-Point or Push-Up Start
  • Rest: full rest between reps — quality over density now
  • Add deeper arcs, 180° cuts, lateral shuffles, reactive starts
  • Add sport loading: basketball dribble through reps, carry equipment
  • Exit when: meets full RTS criteria for that injury
MonitorEvery Session
  • Stopwatch set times every session — log them
  • Symptom and swelling check the following morning
  • Video push-off and landing mechanics each phase
  • Cap sets at 5 reps if mechanics fail under fatigue
  • Target less than 10% time drop-off across a set

Coaching Cues & Mistakes

Cues from Hansen directly

"Arms, rhythm, relaxation." — what he cues during reps. Not "gut it out."
"Push the ground away, stay patient standing up." — controls premature upright posture in acceleration.
"Smooth in, controlled out." — deceleration discipline.
"Non-negotiables in training mean sticking to the basics that create long-term adaptations." — Derek Hansen
"The simplicity of the 10×10 is its power — whether you're rehabbing or working on performance, the principles remain the same." — Derek Hansen
"In return-to-play scenarios, the 10×10 ensures that athletes aren't just working hard, but working on the right things." — Derek Hansen

Common Mistakes

  1. Starting running before the strength gate is met — especially ACL quad symmetry. Number one error in your population.
  2. Chasing top speed too early — short-to-long exists for a reason. Distance jumps of 5m max.
  3. Adding gadgets and variations too early — sleds, vests, creative twists. Get clean reps in first.
  4. Ignoring fatigue-driven mechanic breakdown — if reps 7–10 look terrible, reduce to 5 reps per set.
  5. Skipping arc runs on ankle and knee — going straight from linear to cutting is how re-injuries happen.
  6. Using the calendar as the trigger — "it's been 12 weeks" is not clearance. Criteria-based progression only.
  7. Over-complicating the structure — 10×10 on a marked surface is enough. Resist the urge to innovate when the basics aren't nailed yet.
  8. Not using video — you cannot catch compensation patterns in real time at speed. Film and review.

Key Takeaways

  1. The 10×10 is a dial, not a drill. Same structure runs rehab, conditioning, or speed by changing work-to-rest ratio.
  2. Short accelerations are protective. Low eccentric tissue load + high volume is the mechanism. ~70–75% max velocity is meaningful for sport without the injury risk of higher velocities.
  3. Short-to-long is the engine. Lengthen distance to raise velocity in controlled steps: 10 → 15 → 20 → 30–40m.
  4. Start type = your load dial within the 10m. High Falling → Low Falling → Three-Point → Push-Up. Progress this across the rehab block.
  5. The high-low framework governs weekly structure. Conditioning-dial 10×10 sits in the low zone — recoverable on consecutive days. Speed-dial sits in high — needs 48–72h recovery.
  6. Posture changes per injury. Knee/hamstring = acceleration posture. Calf/Achilles/ankle/hip = upright rolling start.
  7. The gate is everything for your population. ACL: quad symmetry gate (≥70% to run, 90%+ to cut, 9-month minimum to RTS). Ankle: swelling/ROM/hop clearance + arc running before cutting.
  8. Monitor reproducibility. Stable set times = ready to progress. Rapid drop-off = not yet. Target <10% time decay across a set.
  9. Use the time-governor in rehab. Start at 50% slower than their best 10m time and shave off 5–10% per session.
  10. Protect the simplicity. The boredom is the medicine. 3 sets of 10×10 done consistently beats 10 complicated modifications done once.
